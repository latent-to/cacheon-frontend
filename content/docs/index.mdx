**Cacheon** is Bittensor Subnet 14 (SN14). It is an open, on-chain competition for **production-grade inference optimization** of large language models.

Miners submit a **containerized inference server** that serves a fixed model via an OpenAI-compatible API. Validators run every submission against a vLLM baseline on the same hardware and score it on **time-to-first-token**, **throughput**, and **output correctness**. The best server takes the throne; all emission flows to the king until someone beats them.

<Callout title="TL;DR">
  - **Model**: Qwen2.5-72B-Instruct on 4x H200 (fixed topology) - **Submit**: build a Docker image
  that serves the model via `/v1/chat/completions`, commit the image reference on-chain - **Score**:
  `0.5 x TTFT_improvement + 0.5 x throughput_improvement` (floored at 0), with a 99%+ greedy token
  match correctness gate. See [Scoring](./evaluation/scoring). - **Incentive**: king-of-the-hill,
  winner takes all emission
</Callout>

## Why Cacheon exists

Inference has been the bottleneck for serving LLMs at scale. Latency, memory, and cost determine whether a model is usable in production. There is no shortage of optimization techniques (FlashAttention, PagedAttention, speculative decoding, KV cache quantization, custom kernels) but they are benchmarked in isolation and results are impossible to compare fairly.

Cacheon turns the problem into a live, reproducible contest: same model, same hardware, same prompts, same scoring. Miners are free to use any technique, any language, any runtime. The only thing that matters is serving the model faster while keeping outputs correct.

See [Why it matters](/docs/concepts/why-it-matters) for the full problem statement.

## What the subnet is not

- **Not a training subnet.** Weights are fixed at `Qwen2.5-72B-Instruct`. You do not finetune.
- **Not a generic hosting platform.** You serve one specific model. The competition is about serving it faster than everyone else.
- **Not a free trial service.** Miners are limited to one commitment per hotkey. Test locally before you register.
